# Zolve LifeSim MVP

A React + Vite + Tailwind app built from your existing `zolve_life_sim_hackathon_mvp.jsx` UI.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

1. Push this repository to GitHub.
2. Connect GitHub to Vercel or Netlify.
3. Use the default build command: `npm run build`.
4. Use the default publish directory: `dist`.

## Notes

- The app uses `framer-motion`, `lucide-react`, and Tailwind CSS.
- If you deploy to Vercel, no extra build config is required.
